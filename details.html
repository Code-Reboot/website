Details

Dates

The camp will be hosted August 21 - 25, 2017. Each day will span from 1:00 - 5:00.

Age Requirements

Our camp allows all children from ages 8-11.

Hardware Requirements

Each student will be required to bring a laptop. The laptop will need to meet the following requirements:

  • Windows or MacOS
  • 2GB Ram
  • 15GB Free Storage Space

Location

This camp is locally run in Pasadena.

Pricing

We think that too many programming camps are overpriced. That's why we priced Code Reboot 2017 at $100 for the week.
